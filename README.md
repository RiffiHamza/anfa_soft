# Anfa Softworks

Marketing site for **Anfa Softworks LLC** — an independent software studio
building production-grade web, mobile, and AI software. Registered in New
Mexico, USA.

Built with Next.js, React, and Tailwind CSS, exported as a static site.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build

```bash
npm run build    # static export to ./out
```
